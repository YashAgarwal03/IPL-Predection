# 🏏 IPL Points Table Predictor

A dynamic IPL Points Table Predictor built using Flask, Python, HTML, CSS, and JavaScript.

Users can:

- Select Team 1
- Select Team 2
- Select Match Winner
- Handle No Result matches
- Automatically update IPL points table
- Auto sort teams by points

---

# 🚀 Features

✅ Dynamic IPL Points Table  
✅ All 10 IPL Teams  
✅ Match Winner Selection  
✅ No Result Option  
✅ Automatic Table Updates  
✅ Auto Sorting by Points

# ⚙️ Installation

## 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git
```

---

## 2. Open Project Folder

```bash
cd YOUR_REPOSITORY_NAME
```

---

## 3. Create Virtual Environment

```bash
python -m venv venv
```

---

## 4. Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Mac/Linux

```bash
source venv/bin/activate
```

---

## 5. Install Flask

```bash
pip install flask
```

---

# ▶️ Run Project

```bash
python app.py
```

Open browser:

```plaintext
http://127.0.0.1:5000
```
